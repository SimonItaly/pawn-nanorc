# pawn-nanorc
Pawn Syntax for nano

Basically a ripoff of _c.nanorc_ with some keywords added/removed.

I use nano for fast prototyping on Debian.

Copy this to
> /usr/share/nano

and add
> include "/usr/share/nano/pwn.nanorc"

at the end of /usr/nanorc and you're done.
